# <a name="functional-application-consultant-learning-catalog"></a>Funkcionális alkalmazási tanácsadó – képzési katalógus

Ön egy üzleti tevékenységi terület implementációs szakértője? 

A következő katalógus úgy van összeállítva, hogy az alapvető tudásanyagtól a specifikus területek ismereteiig, az alapoktól a legfelsőbb szintekig tartalmazzon képzési anyagokat. Jelezni fogjuk, ha a tartalom több formában is létezik, hogy az igényeinek leginkább megfelelő képzési formát választhassa.

A munka–feladat elemzési kutatásaink szerint a funkcionális tanácsadóknak az alapvető tartalmakat és a specifikus területekre vonatkozó anyagokat is ismerniük kell.
